## עקרונות עיצוב
- **PrimeNG** עם נושא **Aura** (מודרני, נקי, מקצועי), PrimeFlex לפריסה, PrimeIcons לאייקונים.
- **RTL מלא** ועברית (`dir="rtl"`), פונט עברי נעים (למשל Heebo / Assistant).
- **מובייל-פרנדלי (Responsive)** — הורים/תלמידים יגיעו בעיקר מהטלפון.
- **צבעי סטטוס** ביומן עם `Tag`/`Badge`: מאושר (ירוק) / ממתין (כתום) / בוטל (אדום).
- ניווט עליון פשוט (`Menubar`), פעולות עיקריות בולטות.

## מיפוי רכיבי PrimeNG מרכזיים
| צורך | רכיב PrimeNG |
|------|--------------|
| ניווט עליון | `Menubar` / `Toolbar` |
| דשבורדים (קוביות מידע) | `Card`, `Panel` |
| יומן / לוח שיעורים | `FullCalendar` (אינטגרציית PrimeNG) או `Scheduler` |
| בחירת תאריך/שעה לבקשת שיעור | `DatePicker` (Calendar) |
| רשימת תלמידים / תשלומים | `Table` (DataTable) עם מיון וסינון |
| כרטיס תלמיד / טפסים | `Card` + `InputText`, `InputNumber`, `Textarea` |
| חלונות (סיום שיעור, בקשות) | `Dialog` |
| מתגי נראוּת (תלמיד/הורה) | `ToggleSwitch` / `Checkbox` |
| סטטוסים | `Tag`, `Badge` |
| התראות בממשק | `Toast`, `Message` |
| אישור פעולות (אישור/דחייה) | `ConfirmDialog` |
| ספריית מורות ציבורית | `DataView` (כרטיסיות עם סינון) |
